Coursework for Codecademy Pro Intensive

Codecademy Pro Intensive: Build Web Sites from Scratch is an online, 10-week course that teaches the design and deployment of professional web sites using HTML, CSS, GitHub, and JavaScript.

I am documenting here my progress in the course. I've built this web site myself using the knowledge I've gained of HTML and CSS, and deployed it myself using GitHub.
